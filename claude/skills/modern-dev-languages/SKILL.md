# Guide des Langages et Frameworks Modernes

## IMPORTANT : Nature de ce Skill

Ce skill est une **base de reference**, pas une verite absolue. Les informations sont des points de depart pour la reflexion.

**Regles d'utilisation :**
1. Si ton entrainement (knowledge cutoff) contient des informations plus recentes ou plus precises, **privilegie ton entrainement**
2. Les statistiques et tendances evoluent rapidement - verifier avec WebSearch si necessaire
3. Adapter les recommandations au contexte specifique du projet
4. Ne pas hesiter a contredire ce document si les circonstances le justifient

## Vue d'Ensemble

Ce guide couvre les technologies majeures du developpement moderne :

| Technologie | Domaine Principal | Force Cle |
|-------------|-------------------|-----------|
| **Python** | Data Science, AI/ML, Backend, Scripts | Simplicite, ecosysteme riche |
| **JavaScript** | Web Frontend/Backend, Full-stack | Ubiquite navigateur, npm |
| **TypeScript** | Web (superset JS) | Typage statique, tooling |
| **React** | UI Web (SPA) | Composants, Virtual DOM |
| **React Native** | Mobile cross-platform | Code partage iOS/Android |
| **Rust** | Systems, Performance-critical | Securite memoire sans GC |

## Principes Transversaux

### Polyglot Mindset
Aucun langage ne "domine tout". Chaque technologie a sa niche :
- Python : data-centric, prototypage rapide
- JS/TS + React : interfaces utilisateur
- React Native : mobile cross-platform
- Rust : performance et securite memoire

### Interoperabilite
Ces technologies se completent souvent :
- Frontend React + Backend Python (AI features)
- Modules Rust via WebAssembly dans apps web
- Code partage entre React web et React Native mobile
